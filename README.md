# shopping-app

<h2>Introduction</h2>

React and Redux shopping list web app with Firebase's live database updates.

Example app usign React, Redux, React-Router v3.

<h2>Instalation</h2>

App uses create-react-app for fast webpack and web dev server configuration.

```npm install```

and to start...

```npm start```
